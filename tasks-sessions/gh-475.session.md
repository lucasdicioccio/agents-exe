**Read File Range Tests** (8 tests):
- Read single line and verify content with line numbers

- File not found error
- Out of bounds range handling
- Line number formatting with tabs

**Write File Range Tests** (8 tests):
- Replace single line
- Replace line range
- Prepend to file (head)
- Append to file (tail)
- Write to new files using head/tail
- File not found error when not using head/tail
- Lines written count verification

**Edge Case Tests** (7 tests):
- Read from empty file
- Write to empty file with head
- Write to single line file
- Replace entire file
- Range at end of file
- Overlapping ranges behavior
- Trailing newline preservation

**Multi-Range Tests** (3 tests):
- Replace multiple ranges (bottom-to-top application)
- Multiple ranges with `---` separator
- Mismatched content blocks error
