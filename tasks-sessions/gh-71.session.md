## Trade-offs

**Command Naming**
- `export`/`import`: Standard, intuitive
- `pack`/`unpack`: Archive-centric
- `push`/`pull`: Git-centric
- **Decision**: Use `export`/`import` with format-specific flags

**Default Import Destination**
- Current directory: Explicit, user knows where files go
- Config directory (~/.config/agents-exe): Consistent with --agent-file default
- **Decision**: Default to current directory, allow --to-config-dir flag

**Namespace in Export vs Import**
- Export with namespace: Tags the package
- Import with namespace: Filters incoming
- **Decision**: Support both for flexibility
