## Task
Create new file `src/System/Agents/Tools/SystemToolbox.hs` with:

1. **Trace type** for monitoring system toolbox operations:
   ```haskell
   data Trace
       = SystemInfoRequestedTrace !Text  -- capability name
       | SystemInfoRetrievedTrace !Text !NominalDiffTime  -- capability and duration
       | SystemInfoErrorTrace !Text !Text  -- capability and error
       deriving (Show)
   ```

2. **ToolDescription type** (similar to SqliteToolbox):
   ```haskell
   data ToolDescription = ToolDescription
       { toolDescriptionName :: Text
       , toolDescriptionDescription :: Text
       , toolDescriptionToolboxName :: Text
       }
   ```

3. **Toolbox record** - Runtime state:
   ```haskell
   data Toolbox = Toolbox
       { toolboxName :: Text
       , toolboxDescription :: Text
       , toolboxCapabilities :: [SystemToolCapability]
       , toolboxEnvVarFilter :: Maybe Text
       }
   ```

4. **QueryResult type** for structured results:
   ```haskell
   data QueryResult = QueryResult
       { resultCapability :: Text
       , resultData :: Aeson.Value
       , resultExecutionTime :: NominalDiffTime
       }
   instance ToJSON QueryResult
   ```

5. **QueryError type**:
   ```haskell
   data QueryError
       = CapabilityNotEnabledError !Text
       | SystemInfoError !Text
       deriving (Show, Eq)
   ```

6. **Core functions**:
   - `initializeToolbox :: Tracer IO Trace -> SystemToolboxDescription -> IO (Either String Toolbox)`
   - `executeQuery :: Toolbox -> Text -> IO (Either QueryError QueryResult)` - Execute a query for a specific capability
   - `getCapabilityInfo :: SystemToolCapability -> IO (Text, Aeson.Value)` - Gather info for each capability
   - `formatResults :: QueryResult -> ByteString`

7. **Capability implementations** for Linux systems:
   - **Date**: Current UTC and local time, timezone info
   - **OperatingSystem**: Read from `/etc/os-release`, kernel version from `uname`, architecture
   - **EnvVars**: Filtered environment variables (respecting `envVarFilter` pattern)
   - **RunningUser**: Username, UID, GID, home directory, shell
   - **Hostname**: Machine hostname
   - **WorkingDirectory**: Current working directory
   - **ProcessInfo**: Process ID, parent PID, process name
   - **Uptime**: System uptime from `/proc/uptime`

## Files to create
- `src/System/Agents/Tools/SystemToolbox.hs`

## Dependencies
- `unix` package for POSIX system calls
- `time` package for date/time handling
- `directory` package for working directory
- `process` package for reading system info

## Notes

- All file reads should have timeouts to avoid blocking
- Environment variables should be filtered if a pattern is provided (simple substring match is sufficient)
