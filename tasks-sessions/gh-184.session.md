## Summary

Added a new `cowsay` command to the agents-exe CLI that displays messages with the ASCII-art lambda mascot (from `src/System/Agents/SessionPrint.hs`).

### Files Changed:

1. **New file: `src/System/Agents/CLI/Cowsay.hs`**
   - `CowsayOptions` - Options type with `cowsayMessage` (optional) and `cowsayMaxWidth` (default 40)
   - `defaultLogo` - The ASCII art lambda mascot (matching the one in SessionPrint.hs)
   - `handleCowsay` - Reads from argument or stdin, formats and outputs the cowsay text
   - `formatCowsay` - Formats a message in a speech bubble above the mascot
   - `wrapText` - Wraps text to fit within max width
   - `formatSpeechBubble` - Creates the speech bubble around message lines
   - `formatMascot` - Outputs the mascot with connecting lines

2. **Updated: `agents.cabal`**
   - Added `System.Agents.CLI.Cowsay` to the exposed modules list

3. **Updated: `app/Main.hs`**
   - Added `Cowsay` constructor to the `Command` type
   - Added cowsay to the `Show Command` instance
   - Added `parseCowsayCommand` and `parseCowsayOptions` parsers
   - Added command handler in `runCommand`
   - Fixed a typo: `PostgRESTrace` → `PostgRESTTrace`

### Usage:
```bash
# With argument
agents-exe cowsay "Hello, World!"

# From stdin  
echo "Hello from stdin" | agents-exe cowsay

# With custom width
agents-exe cowsay "A very long message that needs wrapping" --width 30
```

The output follows the classic cowsay format with a speech bubble containing the message above the friendly lambda mascot.
