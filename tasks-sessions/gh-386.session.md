# Enhance Tracer to capture recursive agent conversation traces

Depends-on: #385
Base-branch: gh-384-recursive-agent-calls
Final: false

## Overview

Currently, `turnAgentRuntimeIntoIOTool` uses a no-op tracer that discards all trace information from sub-agent executions. This makes debugging recursive agent calls impossible. This issue enhances the tracing system to properly capture and propagate traces from sub-agent calls to the parent conversation.

## Current Problem

In `src/System/Agents/AgentTree/OneShotTool.hs`:

```haskell
runSubAgent ctx (PromptOtherAgent query) = do
    -- No-op tracer - all sub-agent traces are lost!
    let tracer = Tracer $ const $ pure ()
    
    -- Sub-agent runs but traces go nowhere
    (finalTurnContent, _) <- run convId agentWithQuery session0
```

## Solution

### 1. Add New Trace Types

Add to `System.Agents.Runtime.Trace`:

```haskell
data Trace
    = -- existing traces...
    | AgentTrace_SubAgentCall 
        !AgentSlug      -- Parent agent slug
        !AgentId        -- Parent agent ID
        !ConversationId -- Parent conversation ID
        !AgentSlug      -- Sub-agent slug (being called)
        !ConversationId -- Sub-agent conversation ID
        SubAgentTrace   -- Detailed sub-agent trace
    deriving (Show)

data SubAgentTrace
    = SubAgentStarted ConversationId SessionId
    | SubAgentLLMTrace StepId OpenAI.Trace
    | SubAgentToolTrace StepId ToolTrace
    | SubAgentCompleted Text  -- Final response
    | SubAgentFailed Text     -- Error message
    deriving (Show)
```

### 2. Wire Tracer in `turnAgentRuntimeIntoIOTool`

Modify the signature to accept parent's tracer:

```haskell
turnAgentRuntimeIntoIOTool ::
    SessionStore ->
    LoadedApiKeys ->
    OSAgentNode ->
    AgentSlug ->           -- Caller slug
    AgentId ->             -- Caller ID
    ConversationId ->      -- Parent conversation ID
    Tracer IO Trace ->     -- Parent's tracer (NEW)
    ToolRegistration
```

### 3. Create Contramapped Tracer

```haskell
-- In runSubAgent:
let subAgentTracer = contramap 
        (AgentTrace_SubAgentCall callerSlug callerId parentConvId 
                                 (Base.slug agent) convId)
        parentTracer

-- Pass to completion config:
let completionConfig = OpenAICompletionConfig
        { cfgTracer = contramap 
            (AgentTrace_Conversation (Base.slug agent) (osNodeAgentId node) parentConvId . LLMTrace stepId)
            subAgentTracer
        , ...
        }
```

### 4. Emit Traces at Key Points

```haskell
runSubAgent ctx (PromptOtherAgent query) = do
    -- ... setup ...
    
    -- Emit start trace
    runTracer subAgentTracer (SubAgentStarted convId sessionId)
    
    -- Run agent
    result <- try $ run convId agentWithQuery session0
    
    -- Emit completion/error trace
    case result of
        Right (finalContent, _) -> do
            let response = extractResponseText finalContent.llmResponse
            runTracer subAgentTracer (SubAgentCompleted response)
        Left err -> do
            runTracer subAgentTracer (SubAgentFailed (Text.pack $ show err))
```
