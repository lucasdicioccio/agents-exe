## Overview

Enhance the TUI Help tab with a visually appealing ASCII logo and banner at the top, followed by the existing keyboard shortcuts documentation.

## Motivation

The current Help tab (`src/System/Agents/TUI/Render.hs`, function `renderHelpTab`) displays only a plain text list of keyboard shortcuts. Adding a branded ASCII logo and banner will:
- Create a more polished, professional first impression
- Reinforce project identity within the TUI
- Make the Help tab more visually engaging for users

## Current State

The Help tab is rendered by `renderHelpTab` in `src/System/Agents/TUI/Render.hs`:

```haskell
-- | Render the Help tab content.
renderHelpTab :: TuiState -> Widget N
renderHelpTab st =
    borderWithLabel (txt " Help ") $
        viewport AgentInfoWidget Both $
            vBox $
                map txt (st ^. tuiUI . helpContent)
```

The help content is defined in `src/System/Agents/TUI/Event.hs`:
```haskell
defaultHelpContent :: [Text.Text]
defaultHelpContent =
    [ "Keyboard Shortcuts:"
    , ""
    , "Navigation:"
    , "  Tab          - Cycle focus forward through widgets"
    -- ... more shortcuts
    ]
```
