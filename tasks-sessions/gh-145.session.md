## Overview

This is the final integration issue for the tool manipulation feature set (tracking issue #117). This issue represents the ultimate PR that merges all tool manipulation features from the `gh-117` feature branch into `main`.

## Features Being Merged

This PR integrates three new CLI commands for direct tool manipulation:

1. **`agents-exe toolbox list`** - List available tools without loading an agent
2. **`agents-exe toolbox nest`** - Wrap multiple tools into a single nested bash-compatible tool
3. **`agents-exe toolbox mcp`** - Expose toolboxes directly as MCP servers without agent indirection

## New Modules Added

- `System.Agents.Tools.List` - Tool listing functionality
- `System.Agents.Tools.Nest` - Tool nesting/composition functionality  
- `System.Agents.Tools.McpServer` - Direct MCP server for toolboxes

## CLI Changes

### New Command Structure

```
toolbox                          (new command group)
├── list                         (list tools)
│   ├── --agent-file FILE
│   ├── --tool-dir DIR
│   ├── --mcp-config FILE
│   └── --format [table|json|names]
├── nest                         (create nested tool)
│   ├── --agent-file FILE
│   ├── --output FILE
│   ├── --name NAME
│   ├── --include-tools LIST
│   └── --exclude-tools LIST
└── mcp                          (start MCP server from tools)
    ├── --agent-file FILE
    ├── --tool-dir DIR
    ├── --mcp-toolbox NAME:CONFIG
    ├── --stdio
    └── --port PORT
```
