- **Haskell-side filtering**: The trimming logic is implemented in pure Haskell using `partition` and `sortOn` - no SQL changes needed
- The `trimDoneTasks` function:
  1. Separates done tasks from other tasks

  4. Recombines with other tasks, preserving original order by `taskId`
