### Key Design Decisions

2. **Callback Mechanism** (#359): Introduced `SubAgentSessionConfig` to allow parent agents to receive `OnSessionProgress` callbacks from sub-agents, plus a `turnAgentRuntimeIntoIOToolWithCallbacks` function.

3. **TUI Tree View** (#360, #361): Added `ConversationNode` type for tree representation, `buildConversationTree` for building trees from flat lists, and rendering with expand/collapse support.

### Tradeoffs Documented

- **Backward Compatibility**: JSON parsing uses optional fields for legacy sessions
- **API Complexity**: `SubAgentSessionConfig` pattern keeps the API manageable
- **Performance**: Callbacks are synchronous; consider async if needed
- **UI Complexity**: Tree navigation follows common conventions (arrows + Tab)
