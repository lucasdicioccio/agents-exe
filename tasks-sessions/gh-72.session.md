## Trade-offs

**Tool Metadata Storage**
- Embed ScriptInfo in metadata.json: Faster import, no need to run 'describe'
- Always run 'describe' on import: Validates tool works, always current
- **Decision**: Store metadata.json for speed, but validate with 'describe' on import

**Copy vs. Symlink for Git Tools**
- Copy: Self-contained, works offline, no dependency on git repo
- Symlink: Auto-updates when git repo updates, smaller footprint
- **Decision**: Default to copy, --link flag for symlinks

**Global vs. Per-Agent Tools**
- Global: Easier sharing, versioning complexity
- Per-agent: Self-contained, potential duplication
- **Decision**: Support both, recommend per-agent for reproducibility
