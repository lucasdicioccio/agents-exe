Fix TUI command shortcuts to dump session to markdown (#102)

This commit adds two new keyboard shortcuts to the TUI for exporting
sessions to markdown format:

- Ctrl+m: Dumps the currently focused session to a markdown file named
  <conversation-id>.md in the current directory.

- Ctrl+Shift+m: Displays the currently focused session using an external
  viewer specified by the AGENT_MD_VIEWER environment variable. The
  session is written to a temporary file and passed to the viewer.

new shortcuts for users.
